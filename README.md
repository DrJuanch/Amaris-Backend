Requisitos previos

Asegúrarse de tener instalado lo siguiente antes de ejecutar el proyecto:
.NET 8 o superior
Visual Studio Code o Visual Studio
Postman (opcional, para probar la API)

Instalación

Clona el repositorio:
git clone https://github.com/tu-usuario/backend-repo.git
cd backend-repo
Restaura los paquetes NuGet:
dotnet restore

Ejecución del proyecto

Para ejecutar el backend, usa:
dotnet run

Para ejecutar los tsts, usa: 
dotnet test
